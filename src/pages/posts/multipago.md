**Renovación Web con Integración API para Gestión de Localidades de Cobro**

**Contexto:** Este proyecto surgió de la necesidad del cliente de renovar su sitio web, conservando su identidad visual original, pero incorporando una funcionalidad clave: el desarrollo de un plugin personalizado. Este plugin permite consumir su API y conectarla directamente a la web, facilitando la visualización y gestión de las localidades donde se realizan los cobros.

**Soluciones implementadas:**
- *Diseño fiel a la identidad:* Se mantuvo el estilo visual original del sitio, asegurando continuidad y reconocimiento de marca.
- *Integración API personalizada:* Desarrollo de un plugin que consume la API del cliente, permitiendo mostrar dinámicamente las localidades de cobro disponibles.
- *Optimización de experiencia:* La conexión directa entre la web y la API agiliza la actualización de información, mejorando la experiencia tanto para el equipo interno como para los usuarios finales.

**Enfoque clave:**
- *Funcionalidad adaptada:* El plugin se diseñó para ser intuitivo y eficiente, garantizando que la información de localidades esté siempre actualizada y accesible.

**Resultado:** Una web renovada, con mayor capacidad operativa y una gestión de localidades de cobro optimizada, que simplifica la experiencia para el cliente y su equipo.
